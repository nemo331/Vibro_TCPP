	   Описание установочного файла setup.txt
     Все данные задаются сразу после двоеточия без пробела.(Например:
1. Имя файла:izmer5.dat)
     Нельзя использовать символ двоеточия ':' где-либо в файле кроме тех,
что уже стоят.
      Файл состоит из следующих пунктов:
1. Имя файла:izmer5.dat
   Если нет имени файла, то его надо задавать из командной строки. Одновременно
   из командной строки можно задать имя своего установочного файла, который 
   будет использоваться вместо setup.txt. Но файл setup.txt тоже нужен. В нем 
   указывается путь до своего установочного файла.
   Имя файла указывать обязательно с расширением. Имя файла желательно должно
   состоять из 5 символов. Можно меньше,  но не более 7.
2. Номера каналов(не более 16 символов(0123456789abcdef)):01234567
   Здесь указываются номера каналов с 0 по 15. Количество каналов определяется
   из кол-ва символов. Номера каналов могут повторятся.
3. Частота опроса АЦП на один канал(Гц):200
   По умолчанию 1 Гц
4. Объем блока на канал (не более 32767):4096
   Этот параметр определяет сколько отсчетов по каждому из каналов войдет в
   файл(при многоблочной обработке, когда файл данных слишком большой).
   При одноблочной обработке формируется один блок-файл из первой части файла
   данных, но не более размера файла данных.
5. Путь к каталогу данных:\TC\WORK2\DAR\DATA\
   Путь к каталогу, где содержится файл данных. Туда же записывается
    файл с обработанными данными. Подкаталог файлов данных BIN, подкаталог
    файлов с обработанной информацией OSC.
6. Выдавать значения вибрации?(Y or N):Y
   По умолчанию (N)
7. Выдавать показатели (максим., миним. вибрация и т.д.)?(Y or N):Y
   По умолчанию (N)
8. Отсчет времени начинается с НУЛЯ?(Y or N):Y
   По умолчанию (N)
9. Удалять постоянную составляющую сигнала?(Y or N):N
   По умолчанию (N)
10. Формировать один блок-файл?(Y or N):Y
    По умолчанию (N)
    Если символ 'Y',то формируется один блок-файл из первой части файла данных,
    если этот файл больше, чем указано в п.4.
    Если символ 'N',то формируется несколько блок-файлов, в соответствии с размером
    блока (п.4) и объемом файла данных.
11. Выдавать в файл дату и время?(Y or N):N
    По умолчанию (N)
12. Набор коэффициентов преобразования кодов АЦП(1 2 3 4):4
    1- (коэф. 0.25), 2- БКВ-1(Шлейф.осциллограф)
    3- БКВ-2(На ЭВМ), 4- БКВ-2(Шлейф.осциллограф)
    Если нет номера набора данные выдаются в кодах АЦП.
13. Номер ГА:9
14. Номера датчиков:04 15 24 13 27 14 15 отм
15. Места установки датчиков:КР ГПР КТР ТПР ППР ППВ КВ ОТМ
